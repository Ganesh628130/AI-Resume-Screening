# 🤖 AI Resume Screening System

An AI-powered Resume Screening System designed to automate the resume evaluation process and assist recruiters in identifying suitable candidates efficiently.

## 📌 Project Overview

The AI Resume Screening System uses Machine Learning and Natural Language Processing (NLP) techniques to analyze resumes, extract relevant information, and evaluate candidates based on predefined job requirements.

The system helps reduce manual effort in recruitment by automatically screening resumes and identifying the most relevant candidates.

---

## 🎯 Project Objectives

* Automate the resume screening process.
* Reduce manual recruitment effort.
* Analyze and evaluate candidate resumes efficiently.
* Improve candidate shortlisting accuracy.
* Apply Artificial Intelligence and Machine Learning in recruitment workflows.

## ⚙️ How It Works

1. User uploads a resume.
2. The system extracts resume content.
3. Text preprocessing is performed.
4. Machine Learning algorithms analyze resume data.
5. Candidate skills and qualifications are evaluated.
6. The system generates screening results.
7. Recruiters can review shortlisted candidates.
